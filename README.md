# Trilha de Capacitação - Projeto de Extensão

Bem-vindos, devs! 

Este repositório é a nossa **Fonte Única de Verdade (Single Source of Truth)** para a fase de capacitação do projeto de extensão. Aqui, vocês encontrarão todo o mapa de estudos necessário para nivelar os conhecimentos da equipe e nos preparar para construir software com excelência e padrão de mercado.

## 🗺️ O Nosso Roadmap
O ciclo de vida do nosso projeto está estruturado em 5 grandes fases:
1. **Capacitar:** (📍 *Estamos aqui*) Nivelamento técnico, metodologias e fundamentos.
2. **Avaliar:** Validação do conhecimento adquirido e maturidade da equipe.
3. **Implementar:** Mão na massa, codificação das features e arquitetura.
4. **Validar:** Testes, code review e garantia de qualidade (QA).
5. **Lançar:** Deploy em produção e entrega de valor aos usuários.

## 📖 Como usar este repositório?
* **Somente Leitura:** Vocês não farão commits diretos neste repositório. Ele funciona como o *Syllabus* (guia de estudos) oficial do projeto.
* **Estudo Ativo:** Cada link no sumário abaixo leva a um documento detalhado com conceitos, comandos essenciais e materiais de curadoria técnica.
* **Desafios Práticos:** O aprendizado em TI exige prática. Ao final de alguns módulos, haverá missões. Cumprir essas missões nos repositórios pessoais de vocês garantirá o avanço na trilha (e pontos no nosso sistema de acompanhamento e gamificação das aulas!).
